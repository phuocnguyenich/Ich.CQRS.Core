# Ich.CQRS.Core
CQRS reference application is an airline reservation system that allows travel
agents to book flights and make seating reservations.

## Prerequisites
- .NET Core 3.1
- Visual Studio 2019
- SQL Server

## In this repository
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- CQRS
- Memory cache
- Lazy Injection
- MediatR

## Steps to run
- Simply open the solution, compile, and run :))
